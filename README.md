# Práctica GitHub Issues y Project Board

Este repositorio fue creado para demostrar el uso de GitHub Issues, commits referenciando tareas y un Project Board tipo Kanban.

## Objetivo

Registrar tres bugs o tareas ficticias usando GitHub Issues y moverlas en un tablero Kanban desde To Do hasta Done.
